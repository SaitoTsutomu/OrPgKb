Install
------
### Download&Extract Project ###
Use download button.
### Install IIS ###
### Start Visual Studio ###
You must start by Administrator.
### Compile&Publish ###
### Set Accessibility ###
Add Full Access to IIS_IUSERS at C:\Windows\Temp, C:\inetpub\wwwroot\OrPgKb\App_Data
### Do ###
http://localhost/OrPgKb/
